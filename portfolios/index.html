Stage controller

GUI Dashboard to connect and control multiple linear stages. Possible to add additional hardware such as oscilloscope and signal generator.

I built it to create a GUI template for working with multiple hardwares in a measurement.

Tools used: - MATLAB GUIDE - MATLAB Instrument Control Toolbox
